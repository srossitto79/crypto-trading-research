# Contributing to Axiom

Thanks for your interest in Axiom. Axiom is based on Axiom by Judder.

## Issues & discussion — welcome

Bug reports, questions, feature ideas, and general discussion are very welcome:

- **Issues:** open a GitHub issue on the repository.
- **Community:** [Discord](https://discord.gg/vzSQTneq6a)

## Code contributions — paused for now

**Axiom is not yet accepting external code contributions (pull requests).**

The project is currently maintained by a single author, and a contributor agreement (CLA) is not yet in place. Until one is, pull requests from outside contributors will not be merged — this keeps the project's licensing clean while it finds its footing. It's a sequencing decision, not a judgement of your work.

If there's a change you'd like to see, the best path right now is to **open an issue** describing it.

This policy will be revisited once a contributor agreement is in place. Watch this file for updates.
